# IPTV 播放源说明

本资源为辽宁联通IPTV组播地址，通过抓包获得，仅供学习和研究使用。

## 特别声明
- 本项目不定期更新，所有播放源仅限个人学习和测试，严禁用于商业用途。
- 本项目不存储任何流媒体内容，用户使用本资源所产生的法律责任及后果由用户自行承担。
- 提供的M3U播放列表为辽宁联通内网组播地址，使用前需确保宽带已开通IPTV业务（有说法称开通后关闭也可继续使用，但以实际网络环境为准）。

## 使用方法
### 网络环境要求
- 您需要通过路由器拨号上网，或已在路由器上开启组播转发（如IGMP Proxy/IGMP Snooping），以保证能够正常接收组播流。

### 播放方式
- **电脑端**：可使用 PotPlayer、VLC 等支持M3U播放列表的播放器直接打开 `IPTV.m3u` 文件。
- **安卓设备**：推荐使用 DIYP 影音等支持自定义源的IPTV应用。将 `IPTV.txt` 文件导入应用即可观看（通常通过应用内的“节目管理”或“自定义源”功能加载）。
- **其他设备**：若需要将组播地址转换为单播（例如通过 udpxy 服务），请自行查阅相关教程配置。

## 文件说明
- `IPTV.m3u`：标准M3U格式播放列表，适用于大多数播放器。
- `IPTV.txt`：文本格式节目列表，专为 DIYP 影音等应用设计，可直接导入。

## 注意事项
- 组播流通常只能在局域网内播放，无法在公网直接观看。
- 如遇到播放失败，请检查网络环境及设备是否支持组播，并确认IPTV业务状态。
